## Como clonar o repositório usuários Linux

``` bash
$ # Clone o repositório
$ git clone https://github.com/arnaldojr/ac-embarcados/
$ cd ac-embarcados
$ # instale as dependencias python
$ pip install -r requirements.txt --user
```
